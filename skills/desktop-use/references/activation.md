# Windows 与 Office 激活

## 镜像下载

- [山己几子木](https://msdn.sjjzm.com/)：Windows / Office 原版 ISO 镜像索引，提供阿里云盘、腾讯微云、百度网盘、天翼云、移动云、ed2k、BT 等多种下载方式

## 激活工具

| 工具 | 支持产品 | 平台 | 方式 |
|------|---------|------|------|
| [MAS (Microsoft Activation Scripts)](https://massgrave.dev/) | Windows + Office | Windows | PowerShell 一行命令，支持 HWID 数字激活、Ohook、KMS 等多种方式 |
| [CMWTAT Digital Edition](https://github.com/TGSAN/CMWTAT_Digital_Edition) | 仅 Windows 10/11 | Windows | GUI 工具，专注 HWID 数字许可证激活 |
| [Microsoft Office For MacOS](https://github.com/alsyundawy/Microsoft-Office-For-MacOS/) | 仅 Office | macOS | 提供官方安装包链接和 VL 序列化工具 |
