# 质量检测

## 带宽测试 (iperf3)

目标机器上先启动服务端：

```bash
iperf3 -s
```

客户端测试：

```bash
iperf3 -c <目标IP>       # 测试上传
iperf3 -c <目标IP> -R    # 测试下载
iperf3 -c <目标IP> -P 4  # 并行流
```

## 延迟测试 (mtr)

```bash
sudo mtr -r -c 10 <目标IP> # 测试 10 次延迟
```

## IP 信息检测

访问 <https://browserleaks.com/ip>，检查：

- 出口 IP 地址与地理位置
- DNS 泄漏检测（页面内含 DNS Leak Test）
- WebRTC 泄漏检测

## 家宽 / IP 风险检测

访问 <https://ping0.cc>，检查：

- IP 是否被识别为家宽 / 机房 IP
- IP 风险值评分
- 黑名单检测
- 端口开放情况
